# relogio-pomodoro

- Esta é uma aplicação simples baseada na técnica de pomodoro, para auxiliar o usuário a organizar o seu tempo de estudo.

- O tempo é dividido em sessões de estudo e intervalos

- O usuário pode escolher quantas sessões de estudo ele vai realizar, e pode alterar o tempo das sessões e o tempo dos intervalos

- Ao final de cada sessão e de cada intervalo toca um alarme para notificar o usuário
